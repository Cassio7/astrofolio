# Spyfallout

Inspired by the well known party game **Spyfall** rebuilt from scratch using Astro and TypeScript, to play with my friends.

The goal of the project is to provide a simple, fast and client-side version of the game. No accounts, servers, or databases.

Players enter their names, start a game, and play. Nothing is stored, nothing persists.

The project is entirely in Italian (my first language 🇮🇹).

## How the game works

The game is kinda easy, simple rules but can be tricky.

At the start of each round:

- All players receive the same secret location and a role.
- One or more players are spies and do not know the location.

Through turn-based questions and answers, players try to identify the spy, while the spy attempts to guess the location without being discovered.

## Design and image

All the graphics and images are 100% AI-generated. This project is meant to be fun, not complicated, not overengineered, just vibes xD.
